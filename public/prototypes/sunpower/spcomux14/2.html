Header v2

Better integration of search

This version (v2) provides search functionality that is more pronounced, obvious, and easily accessible.

In addition, this version dramatically changes the overall structure and style of the header. The main menu shifts to the left, giving this more focus; while the peripherals (My SunPower, Phone number, Get Started) are shifted and hidden behind secondary interactions (hover states) — these are items that returning users will use over and over again, so the need for them to always be apparent and blaringly obvious is unnecessary.

Note: This is a preliminary concept, thus the secondary states (main menu level-2/dropdown, My SunPower interaction, tablet+ breakpoints) have not be solved. If this concept is selected for definition, these states will be defined more thoroughly.
